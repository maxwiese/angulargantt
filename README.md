# Raum - Auslastungs - Anzeige

Express und Angular 5 basierte Webapplication um Raumbelegung als Diagram darzustellen

## Voraussetzungen

```
Git

Node Js (min. v8.x)
NPM (-> wird mit Node installiert)
```

## Installation

App herunterladen
```
git clone https://github.com/maxwiese/RoomOccupancyGantt
```
App installieren
```
npm install
```
Angular bauen
```
ng build --prod
```


## Nutzung

App starten
```
node server.js
```
